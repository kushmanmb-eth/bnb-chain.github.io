## Reporting a Vulnerability

The BNB Chain team takes security seriously. If you discover a security vulnerability in this repository, please follow responsible disclosure practices.

### How to Report

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Email the BNB Chain security team at **security@bnbchain.org**.
3. Include a detailed description of the vulnerability, steps to reproduce, and potential impact.
4. For smart contract vulnerabilities in the BNB Chain ecosystem, please use the [Bug Bounty Program](https://bugbounty.bnbchain.org).

### What to Expect

- You will receive an acknowledgment within **48 hours**.
- The team will investigate and keep you informed of the progress.
- Once resolved, a security advisory may be published crediting the reporter (with permission).

## Scope

This repository contains **documentation** for BNB Chain. Security issues relevant here include:

- Malicious scripts injected into documentation pages.
- CI/CD pipeline vulnerabilities that could compromise build artifacts.
- Dependency vulnerabilities in build tooling.

For smart contract or node vulnerabilities, refer to the respective BNB Chain repositories.
